Detailed Description

This class simpler version of cbnTreeBuilder to build a centroid method hierarchical tree from small synthetic tractograms yoielding a random dissimilarity matrix (no thresholding and no discarding outliers) reads a seed voxel coordinates list, builds a centroid hierarchical tree from tractography data and writes output files

core function of this class, includes all the necessary tree building steps

Parameters:
nbLevelthe seed voxel neighborhood level restriction to implement, valid values are: 6, 18, 26 (recommended), 32, 92 and 124
memorythe amount of RAM memory to be used for tractogram cache to speed up building the tree, in GBs (recommended value around 50% of total machine memory)
growTypedefines the type of limit for the initial homogenoeus merging stage. TC_GROWNUM = # of clusters, TC_GROWSIZE = size of clusters, TC_GROWOFF = skip this stage.
baseSizecluster size/number limit to stop the initial homogeneous merging stage. If set to 1 or 0 (default) this stage will be skipped
keepDiscardedflag to keep track of discarded voxels in a special field in the tree file, default value true
