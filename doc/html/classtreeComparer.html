Detailed Description

this class implements the main functionalities required for performing meta-leaf matching across trees and calculating tree comparison algorithms across matched-trees

std::pair< float, float > treeComparer::applyNoiseBaseline ( const float  noiseAlpha = 0)

Calculates the noise level of the trees defined as the matching distance of each base node to the corresponding one of the matched tree this "noise level" is then used to eliminate the hierarchical structure below that level (all granularities higher than that) under the assumption that there is insufficient matching quality to use that information for tree comparison

Parameters:
noiseAlphanoise alpha coefficient. Controls the weight of the matching quality on the noise level. When 0 no noise restriction will be applied, when 1 the npoise level will be the averaged matchting quality level of the contained base nodes.
Returns:
a pair structutre with the number of clusters remaining at the highest granularity level for each tree after application of the noise simulation
