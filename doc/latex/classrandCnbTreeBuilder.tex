\subsection{\-Detailed \-Description}
\-This class simpler version of cbn\-Tree\-Builder to build a centroid method hierarchical tree from small synthetic tractograms yoielding a random dissimilarity matrix (no thresholding and no discarding outliers) reads a seed voxel coordinates list, builds a centroid hierarchical tree from tractography data and writes output files
